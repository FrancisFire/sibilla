\doxysubsection{Detailed Description}
An object responsible for managing simulations. When a new request is received, it is delegated to a \mbox{\hyperlink{interfacequasylab_1_1sibilla_1_1core_1_1simulator_1_1SimulationManager}{Simulation\+Manager}} that is built via \mbox{\hyperlink{interfacequasylab_1_1sibilla_1_1core_1_1simulator_1_1SimulationManagerFactory}{Simulation\+Manager\+Factory}}.
